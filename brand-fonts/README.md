# CourtIQ Brand Fonts

Upload these to Claude Design System → "Upload fonts".

## Files
- **Lexend[wght].ttf** — variable font, weights 100–900 (primary: body + display)
- **BarlowCondensed-Regular/Medium/SemiBold/Bold/ExtraBold.ttf** — secondary display

## Mapping (matches your site's CSS)
- `--font-display` → **Lexend** (fallback: Barlow Condensed)
- `--font-body` → **Lexend** (fallback: Barlow)
